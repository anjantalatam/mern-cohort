# Course Hub

### A course selling app built with MERN Stack

Fork and clone the Repo

#### Starting BE

1. `cd server`
2. run `yarn`
3. copy .env.example to .env.dev
4. Add your Mongo url in .env.dev
5. run `yarn dev`

#### Starting FE

1. From Project Directory `cd client`
2. run `yarn`
3. run `yarn dev`
